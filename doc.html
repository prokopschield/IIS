Implementace

Práci na projektu jsme rozděli do dvou adresářů. Frontend a Backend, které sloužili pro stejnojmenné části projektu. Projekt jsme psali v jazycích Svelte, TypeScript, JavaScript, HTML, CSS, Shell a Dockerfile. Frontend byl tvořen i pomocí CSS frameworku Tailwind a jeho knihovky Daisy UI. Pro vyšší přehlednost je kód rozdělen do několika souborů. Např. funkce zajíštující provoz emailů jsou ve ./backend/email.ts. Největší práci na backendu dal jsitě soubor index.ts, který obsahuje většinu funkciovality.

Databáze

Databáži jsme implemetnovali pomocí programu na generování SQL.

Instalace

Pro jednoduchou instalaci stačí použít priložený skript deploy.sh
  • Vytvořte adresář kde chcete, aby server pro aplikaci běžel
  • Do tohoto složku stáhněte aktuální verzi projektu pomocí příkazu `git clone https://github.com/prokopschield/IIS.git "název_složky"`a následně jděte do nového vytvořeného adresáře a spustě `git pull`
  • Následovně je potřebné spustit kontajnery, proto třeba vypnout všechny eziace, stáhnut závislosti a následně kontajnery spustit 'sudo compose down'
    'cd backend && yarn && tsc' (což nainstaluje potřebné závislosti pro backend)
    'cd prokop-frontend && yarn && yarn build' (který nainstaluje potřebné závislosti pro frontend) 'docker compose up --build $*'

  • Požadavky na SW
  • Rozbalení, konfigurace a inicializace databaze

Známé problémy

Vytvořili jsme dva frontendy. Z nichž doporučujeme použít ./prokop-frontend/. První byl výsledkem dlouhodobé snahy lidí spolu vytvořit funkční frontend pomocí Tailwindu. Druhý je funkčně napojený na backend našeho systému. Systém není dokonale otestován. Nejsou plně implementovány příjmé zprávy.
